Этот проект - бот в телеграме и ВК одновременно. Что он делает?
Помогает пользователю сгенерировать короткий поздравительный видео кружок с участием известных людей.
После входа в бота пользователю предлагается загрузить свой текст поздравления, либо  попросить AI сгенерировать текст поздравления.
Во втором случае необходимо провести опрос пользователя по получателю, как зовут, какая тематика поздравления, хотелось бы отметить какие-либо детали или свои пожелания в свободной форме.
Далее пользователь нажимает кнопку сгенерировать вариант поздравления и ему предлагается текст поздравления на согласование.
Далее если ему понравился текст, то он его подвтерждает . Кнопки Да или Нет.
Если нажимает нет, то цикл генерации повторяется.
Если нажимает Да, то переходим к выбору персонажа и его изображения.
После выбора изображения идет оплата через Юкассу
После успешного подвтерждения платежа запускается процесс 
1. генерации аудио из выбранного текста с помощью сервиса elevenlabs (смотри voice_service.py)
2. из полученного аудио на шаге 1 генерируем видео с помощью нашего собственного сервиса , который находится на сервисе runpod
 пример на питоне

Мы получаем номер jobid и можем отслеживать статус генерации видео 
Для пользователя необходимо сделать возможность посмотреть статус данной генерации в меню телеграм бота
По итогу генерации видео кружка пользователю выдается итоговый результат в телеграм-боте

Стек технологий
- python 3.12
- asynchttp latest
- aiogram latest compatible - библиотека для телеграм бота
- vk api - если есть библиотека от ВК можно ее
- postgresql - база данных
- картинки персонажей на локальном диске
- для юкассы есть либа для Питона и можно тестовый аккаунт создать 
- имитируя оплату
- ИИ для генерации текстов поздравлений - Open AI API через openrouter
- Docker и Docker Compose для локального тестирования, 
- CI/CD и Docker Compose Prod для github actions
- режимы ботов polling для локального тестирования
- webhook для прода
